## GhostRemont 0.3

This is an big update with more fixes and improvements.

* Fixed the hitbox of the cart
* Fixed the mechanics of carrying employees' bodies
* Fixed the installation of the skin for the license
* Ghosts can no longer enter the elevator
* Fixed the collision of the ghost's gaze with the player's hitbox in spectator mode
* Added the ability to move heavy objects while lying down
* Objects are now more visible thanks to visual cues
* Added the ability to restart the game at any time
* Improved English localization (Thanks to RavenFog12)
* Fixed server issues
* Restricted ladder spawning
* Added some settings to the TAB panel
* Fixed the starting elevator boundary
* And other minor bugs...
